
## How It Works

1. User registers or logs in.
2. User reports a lost or found item.
3. Item details are stored in the database.
4. Users can browse, search, and filter listings.
5. Similar items are matched based on category, location, and title.
6. Once recovered, the item can be marked as claimed.
